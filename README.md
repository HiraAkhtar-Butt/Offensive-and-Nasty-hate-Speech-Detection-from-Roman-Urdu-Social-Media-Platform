# Offensive-and-Nasty-hate-Speech-Detection-from-Roman-Urdu-Social-Media-Platform
The research tested the performance of Naïve Bayes, Random Forest, SVM, Logistic Regression, LSTM and MLP over RUHSOLD, VVD-21 and HR-SU-20 datasets. Experiments were carried out at a word-level and a sentence-level. Results were assessed using precision, recall, F1-score and accuracy.
